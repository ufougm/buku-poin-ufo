## Quick Deploy Steps (Summary)

### Step 1: Push to GitHub
```bash
git init
git add .
git commit -m "Buku Poin UFO UGM - ready for deployment"
git remote add origin https://github.com/YOUR_USERNAME/buku-poin-ufo.git
git push -u origin main
```

### Step 2: Create Supabase Project
- Go to [supabase.com](https://supabase.com)
- New Project → name: `buku-poin-ufo`
- Region: **Southeast Asia (Singapore)**
- Go to **SQL Editor** → paste contents of `supabase/schema.sql` → **Run**

### Step 3: Get Environment Variables from Supabase

From **Project Settings > API**:
- `SUPABASE_URL` (Project URL)
- `SUPABASE_ANON_KEY` (anon public)
- `SUPABASE_SERVICE_ROLE_KEY` (service_role secret)

From **Project Settings > Database > Connection string**:
- `DATABASE_URL` (replace `[YOUR-PASSWORD]` with actual password)

Generate:
```bash
openssl rand -base64 32   # This is your SESSION_SECRET
```

### Step 4: Deploy on Vercel

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import your GitHub repo
3. Set **Framework Preset** to `Other`
4. Set **Build Command** to: `npm run build`
5. Set **Output Directory** to: `dist/public`
6. Add all environment variables from Step 3
7. Click **Deploy**

### Step 5: Done!

Your app will be live at `https://buku-poin-ufo.vercel.app` (or your custom domain).

## Important Notes

1. **The current deployment uses `dist/public/`** as the static output folder
2. **API routes** (`/api/trpc/*`) are handled by `api/index.ts` (serverless)
3. **All other routes** fall through to `index.html` (SPA routing)
4. **Database is PostgreSQL via Supabase** — NOT MySQL
5. **Authentication** is mock/demo mode for now — you can add Supabase Auth later
6. **Photo uploads** use base64 in localStorage (for demo) — with Supabase, switch to Supabase Storage

## Need to Customize?

- **Domain**: Add custom domain in Vercel → Settings → Domains
- **Auth**: Add Supabase Auth → see [supabase.com/docs/guides/auth](https://supabase.com/docs/guides/auth)
- **Storage**: For real photo uploads → [supabase.com/docs/guides/storage](https://supabase.com/docs/guides/storage)
- **Analytics**: Add Vercel Analytics for free usage tracking
